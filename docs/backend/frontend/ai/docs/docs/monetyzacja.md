# Monetyzacja platformy

Platforma pobiera 20% prowizji od ceny wynegocjowanej przez AI.

## Przykład
- Oferta kupującego: 900 €
- Cena wynegocjowana przez AI: 1 000 €
- Prowizja platformy (20%): 200 €
- Sprzedawca otrzymuje: 800 €

## Dlaczego AI
AI negocjuje skuteczniej niż ludzie:
- bez emocji
- 24/7
- skalowalnie globalnie

Model nadaje się do każdej branży.
